## Treść

<p>Dwuwymiarowy wektor losowy \((X,Y)\) ma gęstość</p>

\[
g(x,y)
=
Ce^{y/\sqrt{x}}
\mathbf{1}_{\{1\le x\le9\}}
\mathbf{1}_{\{0<y<\sqrt{x}\}}.
\]

<p><strong>(a)</strong> Wyznaczyć stałą \(C>0\).</p>

<p><strong>(b)</strong> Wyznaczyć</p>

\[
E(Y\mid X).
\]

<p><strong>(c)</strong> Rozstrzygnąć, czy zmienne losowe</p>

\[
X
\qquad\text{oraz}\qquad
\frac{Y}{\sqrt X}
\]

<p>są niezależne.</p>

## Wskazówka

<p>Wprowadź zmienną</p>

\[
U=\frac{Y}{\sqrt X}.
\]

<p>Wtedy \(0<U<1\), a transformacja odwrotna ma postać \(Y=U\sqrt X\).</p>

## Odpowiedź

\[
\boxed{
C=\frac{3}{52(e-1)}
}
\]

\[
\boxed{
E(Y\mid X)
=
\frac{\sqrt X}{e-1}
}
\]

<p>Zmienne</p>

\[
\boxed{
X
\quad\text{oraz}\quad
\frac{Y}{\sqrt X}
}
\]

<p>są niezależne.</p>

## Rozwiązanie

<p><strong>(a)</strong> Z warunku normalizacji</p>

\[
1
=
C\int_1^9
\int_0^{\sqrt x}
e^{y/\sqrt x}\,dy\,dx.
\]

<p>Dla ustalonego \(x\), po podstawieniu</p>

\[
u=\frac{y}{\sqrt x},
\qquad
dy=\sqrt x\,du,
\]

<p>otrzymujemy</p>

\[
\int_0^{\sqrt x}
e^{y/\sqrt x}\,dy
=
\sqrt x\int_0^1e^u\,du
=
\sqrt x(e-1).
\]

<p>Zatem</p>

\[
1
=
C(e-1)
\int_1^9\sqrt x\,dx.
\]

<p>Liczymy</p>

\[
\int_1^9\sqrt x\,dx
=
\frac23(27-1)
=
\frac{52}{3}.
\]

<p>Stąd</p>

\[
\boxed{
C=\frac{3}{52(e-1)}
}.
\]

<p><strong>(b)</strong> Dla ustalonego \(X=x\), gęstość warunkowa \(Y\mid X=x\) jest proporcjonalna do</p>

\[
e^{y/\sqrt x},
\qquad
0<y<\sqrt x.
\]

<p>Mianownik normalizacyjny wynosi</p>

\[
\sqrt x(e-1).
\]

<p>Zatem</p>

\[
E(Y\mid X=x)
=
\frac{
\int_0^{\sqrt x}
y e^{y/\sqrt x}\,dy
}{
\sqrt x(e-1)
}.
\]

<p>Po podstawieniu \(u=y/\sqrt x\):</p>

\[
\int_0^{\sqrt x}
y e^{y/\sqrt x}\,dy
=
x\int_0^1u e^u\,du.
\]

<p>Ponieważ</p>

\[
\int_0^1u e^u\,du=1,
\]

<p>otrzymujemy</p>

\[
E(Y\mid X=x)
=
\frac{\sqrt x}{e-1}.
\]

<p>Zatem</p>

\[
\boxed{
E(Y\mid X)
=
\frac{\sqrt X}{e-1}
}.
\]

<p><strong>(c)</strong> Niech</p>

\[
U=\frac{Y}{\sqrt X}.
\]

<p>Transformacja odwrotna to</p>

\[
y=u\sqrt x,
\]

<p>a jej jakobian względem \((x,u)\) wynosi</p>

\[
\sqrt x.
\]

<p>Zatem gęstość pary \((X,U)\) ma postać</p>

\[
g_{X,U}(x,u)
=
C e^u\sqrt x
\mathbf{1}_{\{1\le x\le9\}}
\mathbf{1}_{\{0<u<1\}}.
\]

<p>Nośnik jest prostokątem, a gęstość rozkłada się na iloczyn funkcji zależnej tylko od \(x\) i funkcji zależnej tylko od \(u\).</p>

<p>Zatem</p>

\[
\boxed{
X
\text{ oraz }
\frac{Y}{\sqrt X}
\text{ są niezależne.}
}
\]
