## Treść

<p>Zmienna losowa \(X\) przyjmuje wartości w zbiorze</p>

\[
\{0,1,2,\ldots\}
\]

<p>i dla każdej funkcji</p>

\[
f:\{0,1,\ldots\}\to\mathbb R
\]

<p>zachodzi</p>

\[
E[Xf(X)]
=
5E[f(X+1)].
\]

<p>Wykazać, że rozkład zmiennej losowej \(X\) jest jednoznacznie wyznaczony przez tę relację, wyznaczyć ten rozkład oraz obliczyć</p>

\[
E[(X+1)^2\mid X>0].
\]

## Wskazówka

<p>Wstaw jako \(f\) funkcję wskaźnikową pojedynczego punktu. Otrzymasz rekurencję na prawdopodobieństwa</p>

\[
p_k=P(X=k).
\]

## Odpowiedź

\[
\boxed{
X\sim\operatorname{Poisson}(5)
}
\]

<p>oraz</p>

\[
\boxed{
E[(X+1)^2\mid X>0]
=
\frac{41-e^{-5}}{1-e^{-5}}
}
\]

## Rozwiązanie

<p>Oznaczmy</p>

\[
p_k=P(X=k).
\]

<p>Dla \(k\ge1\) wybieramy funkcję</p>

\[
f(x)=\mathbf{1}_{\{x=k\}}.
\]

<p>Wtedy</p>

\[
E[Xf(X)]
=
kp_k.
\]

<p>Z drugiej strony</p>

\[
E[f(X+1)]
=
P(X+1=k)
=
p_{k-1}.
\]

<p>Zatem</p>

\[
kp_k=5p_{k-1}.
\]

<p>Czyli</p>

\[
p_k=\frac5k p_{k-1}.
\]

<p>Iterując, otrzymujemy</p>

\[
p_k
=
\frac{5^k}{k!}p_0.
\]

<p>Ponieważ suma prawdopodobieństw wynosi \(1\),</p>

\[
1
=
\sum_{k=0}^{\infty}p_k
=
p_0
\sum_{k=0}^{\infty}
\frac{5^k}{k!}
=
p_0e^5.
\]

<p>Stąd</p>

\[
p_0=e^{-5}
\]

<p>oraz</p>

\[
\boxed{
p_k=e^{-5}\frac{5^k}{k!}.
}
\]

<p>Jest to dokładnie rozkład Poissona z parametrem \(5\), a rekurencja wraz z normalizacją pokazuje jego jednoznaczność.</p>

<p>Teraz liczymy</p>

\[
E[(X+1)^2\mid X>0].
\]

<p>Dla \(X\sim\operatorname{Poisson}(5)\)</p>

\[
E[X]=5
\]

<p>oraz</p>

\[
E[X^2]
=
\operatorname{Var}(X)+E[X]^2
=
5+25
=
30.
\]

<p>Zatem</p>

\[
E[(X+1)^2]
=
E[X^2]+2E[X]+1
=
30+10+1
=
41.
\]

<p>Przy \(X=0\) wartość \((X+1)^2\) wynosi \(1\), więc</p>

\[
E[(X+1)^2\mathbf{1}_{\{X>0\}}]
=
41-e^{-5}.
\]

<p>Ponadto</p>

\[
P(X>0)=1-e^{-5}.
\]

<p>Stąd</p>

\[
\boxed{
E[(X+1)^2\mid X>0]
=
\frac{41-e^{-5}}{1-e^{-5}}.
}
\]
