## Treść

<p>Niech \(a\ge b>0\). Z prostokąta \(P\) o bokach długości \(a\) i \(b\) losujemy niezależnie punkty</p>

\[
A_1,\ldots,A_n
\]

<p>zgodnie z rozkładem jednostajnym na \(P\). Niech \(D_i\) oznacza odległość punktu \(A_i\) od brzegu prostokąta.</p>

<p>Definiujemy</p>

\[
X=\min\{D_1,\ldots,D_n\},
\qquad
Y=\max\{D_1,\ldots,D_n\}.
\]

<p><strong>(a)</strong> Wyznaczyć dystrybuanty i gęstości zmiennych losowych \(X\) i \(Y\).</p>

<p><strong>(b)</strong> Obliczyć</p>

\[
P\left(
Y\le\frac a4
\;\middle|\;
X\ge\frac b8
\right).
\]

## Wskazówka

<p>Dla pojedynczego punktu \(A_i\) zdarzenie \(D_i>t\) oznacza, że punkt leży w prostokącie odsuniętym od każdego boku o \(t\), czyli w prostokącie o bokach \(a-2t\) i \(b-2t\).</p>

## Odpowiedź

<p>Niech</p>

\[
S(t)
=
\frac{(a-2t)(b-2t)}{ab},
\qquad
0\le t\le\frac b2.
\]

<p>Wtedy</p>

\[
\boxed{
F_X(t)=1-S(t)^n
}
\]

<p>oraz</p>

\[
\boxed{
F_Y(t)=\bigl(1-S(t)\bigr)^n
}
\]

<p>dla \(0\le t\le b/2\), z naturalnym przedłużeniem poza ten przedział.</p>

<p>Gęstości:</p>

\[
\boxed{
f_X(t)
=
nS(t)^{n-1}
\frac{2(a+b-4t)}{ab}
}
\]

<p>oraz</p>

\[
\boxed{
f_Y(t)
=
n\bigl(1-S(t)\bigr)^{n-1}
\frac{2(a+b-4t)}{ab}
}
\]

<p>dla \(0<t<b/2\).</p>

<p>W punkcie (b):</p>

\[
\boxed{
P\left(
Y\le\frac a4
\;\middle|\;
X\ge\frac b8
\right)
=
1
}
\]

<p>gdy \(a\ge2b\), natomiast dla \(b\le a<2b\)</p>

\[
\boxed{
\left(
\frac{
S(b/8)-S(a/4)
}{
S(b/8)
}
\right)^n
}.
\]

## Rozwiązanie

<p>Dla pojedynczego punktu \(A_i\) i \(0\le t\le b/2\) zdarzenie</p>

\[
D_i>t
\]

<p>oznacza, że punkt znajduje się co najmniej w odległości \(t\) od każdego boku prostokąta.</p>

<p>Dozwolony obszar jest więc prostokątem o bokach</p>

\[
a-2t
\qquad\text{oraz}\qquad
b-2t.
\]

<p>Zatem</p>

\[
P(D_i>t)
=
\frac{(a-2t)(b-2t)}{ab}
=
S(t).
\]

<p><strong>Rozkład minimum \(X\).</strong></p>

<p>Mamy</p>

\[
P(X>t)
=
P(D_1>t,\ldots,D_n>t)
=
S(t)^n,
\]

<p>ponieważ punkty są losowane niezależnie.</p>

<p>Stąd</p>

\[
F_X(t)
=
P(X\le t)
=
1-S(t)^n.
\]

<p>Po zróżniczkowaniu</p>

\[
S'(t)
=
-\frac{2(a+b-4t)}{ab}.
\]

<p>Zatem</p>

\[
\boxed{
f_X(t)
=
nS(t)^{n-1}
\frac{2(a+b-4t)}{ab}
}.
\]

<p><strong>Rozkład maksimum \(Y\).</strong></p>

<p>Mamy</p>

\[
P(Y\le t)
=
P(D_1\le t,\ldots,D_n\le t)
=
\bigl(1-S(t)\bigr)^n.
\]

<p>Stąd</p>

\[
\boxed{
F_Y(t)
=
\bigl(1-S(t)\bigr)^n
}
\]

<p>oraz</p>

\[
\boxed{
f_Y(t)
=
n\bigl(1-S(t)\bigr)^{n-1}
\frac{2(a+b-4t)}{ab}
}.
\]

<p><strong>(b)</strong> Warunek</p>

\[
X\ge\frac b8
\]

<p>oznacza, że każdy z punktów ma odległość od brzegu co najmniej \(b/8\).</p>

<p>Zdarzenie</p>

\[
Y\le\frac a4
\]

<p>oznacza, że każdy punkt ma odległość od brzegu nie większą niż \(a/4\).</p>

<p>Jeżeli</p>

\[
a\ge2b,
\]

<p>to</p>

\[
\frac a4\ge\frac b2,
\]

<p>a ponieważ zawsze \(D_i\le b/2\), warunek \(Y\le a/4\) zachodzi automatycznie. Zatem prawdopodobieństwo warunkowe wynosi \(1\).</p>

<p>Jeżeli natomiast</p>

\[
b\le a<2b,
\]

<p>to dla pojedynczego punktu prawdopodobieństwo warunku</p>

\[
\frac b8\le D_i\le\frac a4
\]

<p>wynosi</p>

\[
S\left(\frac b8\right)
-
S\left(\frac a4\right).
\]

<p>Ponieważ punkty są niezależne,</p>

\[
P\left(
Y\le\frac a4,\,
X\ge\frac b8
\right)
=
\left[
S\left(\frac b8\right)
-
S\left(\frac a4\right)
\right]^n.
\]

<p>Natomiast</p>

\[
P\left(
X\ge\frac b8
\right)
=
S\left(\frac b8\right)^n.
\]

<p>Zatem</p>

\[
\boxed{
P\left(
Y\le\frac a4
\;\middle|\;
X\ge\frac b8
\right)
=
\left(
\frac{
S(b/8)-S(a/4)
}{
S(b/8)
}
\right)^n
}.
\]
