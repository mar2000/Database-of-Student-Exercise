## Treść

<p>W urnie znajduje się \(100\) ponumerowanych kul. Losujemy bez zwracania \(25\) spośród nich.</p>

<p>Niech \(X\) oznacza największy wylosowany numer. Wyznaczyć rozkład \(X\).</p>

## Wskazówka

<p>Zdarzenie \(X\le k\) oznacza, że wszystkie \(25\) wylosowanych kul ma numery należące do zbioru \(\{1,\ldots,k\}\). Dla rozkładu punktowego ustal wartość największego numeru i wybierz pozostałe \(24\) kule spośród mniejszych numerów.</p>

## Odpowiedź

<p>Dla \(k=25,\ldots,100\):</p>

\[
\boxed{
P(X=k)
=
\frac{\binom{k-1}{24}}
{\binom{100}{25}}
}
\]

<p>oraz dystrybuanta</p>

\[
\boxed{
F_X(k)
=
P(X\le k)
=
\frac{\binom{k}{25}}
{\binom{100}{25}}
}
\]

<p>dla \(25\le k\le100\).</p>

## Rozwiązanie

<p>Każdy \(25\)-elementowy podzbiór zbioru numerów</p>

\[
\{1,2,\ldots,100\}
\]

<p>jest jednakowo prawdopodobny. Liczba wszystkich możliwych wyników wynosi</p>

\[
\binom{100}{25}.
\]

<p>Aby \(X=k\), kula o numerze \(k\) musi zostać wylosowana, natomiast pozostałe \(24\) kule muszą mieć numery mniejsze od \(k\).</p>

<p>Możemy je wybrać spośród</p>

\[
1,\ldots,k-1
\]

<p>na</p>

\[
\binom{k-1}{24}
\]

<p>sposobów.</p>

<p>Zatem dla \(k=25,\ldots,100\)</p>

\[
\boxed{
P(X=k)
=
\frac{\binom{k-1}{24}}
{\binom{100}{25}}
}.
\]

<p>Równoważnie, zdarzenie \(X\le k\) oznacza, że wszystkie \(25\) kul zostały wybrane spośród pierwszych \(k\) numerów. Zatem</p>

\[
\boxed{
P(X\le k)
=
\frac{\binom{k}{25}}
{\binom{100}{25}}
}
\]

<p>dla \(25\le k\le100\), przy czym dla \(k<25\) dystrybuanta jest równa \(0\), a dla \(k\ge100\) równa \(1\).</p>
