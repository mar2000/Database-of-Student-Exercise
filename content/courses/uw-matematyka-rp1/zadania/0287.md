## Treść

<p>Wektor losowy \((X,Y)\) ma gęstość</p>

\[
g(x,y)
=
Ce^{-x}y\mathbf{1}_{\{0\le y\le x\}}.
\]

<p><strong>(a)</strong> Wyznaczyć stałą \(C\).</p>

<p><strong>(b)</strong> Obliczyć</p>

\[
P(2Y\le X).
\]

<p><strong>(c)</strong> Obliczyć</p>

\[
E\left((X+Y)^2\mid X\right).
\]

## Wskazówka

<p>Dla ustalonego \(X=x\), gęstość warunkowa \(Y\mid X=x\) jest proporcjonalna do \(y\) na przedziale \([0,x]\).</p>

## Odpowiedź

\[
\boxed{C=1}
\]

\[
\boxed{
P(2Y\le X)=\frac14
}
\]

\[
\boxed{
E\left((X+Y)^2\mid X\right)
=
\frac{17}{6}X^2
}
\]

## Rozwiązanie

<p><strong>(a)</strong> Z warunku normalizacji</p>

\[
1
=
C\int_0^\infty
\int_0^x
e^{-x}y\,dy\,dx.
\]

<p>Całka wewnętrzna wynosi</p>

\[
e^{-x}\frac{x^2}{2}.
\]

<p>Zatem</p>

\[
1
=
\frac C2
\int_0^\infty x^2e^{-x}\,dx.
\]

<p>Ponieważ</p>

\[
\int_0^\infty x^2e^{-x}\,dx
=
2,
\]

<p>otrzymujemy</p>

\[
\boxed{C=1}.
\]

<p><strong>(b)</strong></p>

\[
P(2Y\le X)
=
\int_0^\infty
\int_0^{x/2}
e^{-x}y\,dy\,dx.
\]

<p>Całka wewnętrzna daje</p>

\[
e^{-x}\frac{x^2}{8}.
\]

<p>Stąd</p>

\[
P(2Y\le X)
=
\frac18
\int_0^\infty x^2e^{-x}\,dx
=
\boxed{\frac14}.
\]

<p><strong>(c)</strong> Gęstość brzegowa \(X\) wynosi</p>

\[
f_X(x)
=
\int_0^x e^{-x}y\,dy
=
e^{-x}\frac{x^2}{2}.
\]

<p>Zatem dla \(0\le y\le x\)</p>

\[
f_{Y\mid X=x}(y)
=
\frac{e^{-x}y}{e^{-x}x^2/2}
=
\frac{2y}{x^2}.
\]

<p>Liczymy</p>

\[
E(Y\mid X=x)
=
\int_0^x
y\frac{2y}{x^2}\,dy
=
\frac{2x}{3},
\]

<p>oraz</p>

\[
E(Y^2\mid X=x)
=
\int_0^x
y^2\frac{2y}{x^2}\,dy
=
\frac{x^2}{2}.
\]

<p>Wobec tego</p>

\[
E((X+Y)^2\mid X=x)
=
x^2
+
2x\frac{2x}{3}
+
\frac{x^2}{2}.
\]

<p>Po uproszczeniu</p>

\[
E((X+Y)^2\mid X=x)
=
\frac{17}{6}x^2.
\]

<p>Zatem</p>

\[
\boxed{
E\left((X+Y)^2\mid X\right)
=
\frac{17}{6}X^2
}.
\]
