## Treść

<p>Zmienne losowe</p>

\[
X_1,X_2,\ldots
\]

<p>są niezależne i mają wspólną gęstość</p>

\[
f(x)=3x^2\mathbf{1}_{[0,1]}(x).
\]

<p><strong>(a)</strong> Zbadać zbieżność prawie na pewno ciągu</p>

\[
S_n
=
\frac{X_1+\cdots+X_n}{n}.
\]

<p><strong>(b)</strong> Dla jakich wartości parametru \(a\) ciąg</p>

\[
Z_n
=
n^{1/3}(S_n-a)
\]

<p>jest zbieżny w \(L^2\)?</p>

## Wskazówka

<p>Oblicz \(E[X_1]\) i \(\operatorname{Var}(X_1)\). W punkcie (b) rozważ osobno przypadek \(a=E[X_1]\) i \(a\ne E[X_1]\).</p>

## Odpowiedź

<p><strong>(a)</strong></p>

\[
\boxed{
S_n\longrightarrow\frac34
\quad\text{prawie na pewno}
}
\]

<p><strong>(b)</strong> Ciąg \(Z_n\) jest zbieżny w \(L^2\) dokładnie dla</p>

\[
\boxed{
a=\frac34
}
\]

<p>i wtedy</p>

\[
\boxed{
Z_n\longrightarrow0
\quad\text{w }L^2.
}
\]

## Rozwiązanie

<p>Obliczamy wartość oczekiwaną:</p>

\[
E[X_1]
=
\int_0^1x\cdot3x^2\,dx
=
3\int_0^1x^3\,dx
=
\frac34.
\]

<p>Ponadto</p>

\[
E[X_1^2]
=
3\int_0^1x^4\,dx
=
\frac35.
\]

<p>Zatem</p>

\[
\operatorname{Var}(X_1)
=
\frac35-\left(\frac34\right)^2
=
\frac{3}{80}.
\]

<p><strong>(a)</strong> Z silnego prawa wielkich liczb</p>

\[
\boxed{
S_n\longrightarrow\frac34
\quad\text{prawie na pewno}.
}
\]

<p><strong>(b)</strong> Mamy</p>

\[
E[Z_n]
=
n^{1/3}\left(\frac34-a\right).
\]

<p>Jeżeli \(a\ne3/4\), to wartość bezwzględna tej średniej dąży do nieskończoności, więc ciąg nie może być zbieżny w \(L^2\).</p>

<p>Niech teraz</p>

\[
a=\frac34.
\]

<p>Wtedy \(E[Z_n]=0\), a</p>

\[
E[Z_n^2]
=
n^{2/3}\operatorname{Var}(S_n).
\]

<p>Ponieważ</p>

\[
\operatorname{Var}(S_n)
=
\frac{1}{n}\operatorname{Var}(X_1)
=
\frac{3}{80n},
\]

<p>otrzymujemy</p>

\[
E[Z_n^2]
=
\frac{3}{80n^{1/3}}
\longrightarrow0.
\]

<p>Zatem</p>

\[
\boxed{
Z_n\longrightarrow0
\quad\text{w }L^2
}
\]

<p>dokładnie dla</p>

\[
\boxed{a=\frac34}.
\]
