## Treść

<p>Niech</p>

\[
w=(w_1,w_2,\ldots,w_k)\in\mathbb{R}^k
\]

<p>będzie ustalonym wektorem. Niech</p>

\[
f(x)=e^{-w\cdot x}
\]

<p>i niech</p>

\[
A=[0,\infty)^k.
\]

<p>Obliczyć</p>

\[
\int_A f\,d\lambda_k.
\]

## Wskazówka

<p>Zapisz funkcję wykładniczą jako iloczyn</p>

\[
e^{-w\cdot x}
=
\prod_{i=1}^k e^{-w_i x_i}
\]

<p>i zastosuj twierdzenie Tonellego. Następnie rozpatrz osobno przypadek \(w_i>0\) dla wszystkich \(i\) oraz przypadek, gdy dla pewnego \(i\) zachodzi \(w_i\le0\).</p>

## Odpowiedź

\[
\int_A e^{-w\cdot x}\,d\lambda_k(x)
=
\begin{cases}
\dfrac{1}{w_1w_2\cdots w_k},
&
\text{gdy } w_i>0 \text{ dla każdego } i,
\\[8pt]
+\infty,
&
\text{gdy } w_i\le0 \text{ dla pewnego } i.
\end{cases}
\]

## Rozwiązanie

<p>Mamy</p>

\[
f(x)
=
e^{-w\cdot x}
=
e^{-(w_1x_1+\cdots+w_kx_k)}.
\]

<p>Ponadto</p>

\[
A=[0,\infty)^k.
\]

<p>Funkcja \(f\) jest nieujemna i mierzalna, więc możemy zastosować twierdzenie Tonellego.</p>

<p><strong>1. Rozdzielenie funkcji na iloczyn</strong></p>

<p>Zauważmy, że</p>

\[
e^{-(w_1x_1+\cdots+w_kx_k)}
=
\prod_{i=1}^k e^{-w_i x_i}.
\]

<p>Zatem</p>

\[
\int_A f\,d\lambda_k
=
\int_{[0,\infty)^k}
\prod_{i=1}^k e^{-w_i x_i}
\,dx_1\cdots dx_k.
\]

<p>Na mocy twierdzenia Tonellego możemy zapisać tę całkę jako iloczyn całek jednowymiarowych:</p>

\[
\int_A f\,d\lambda_k
=
\prod_{i=1}^k
\int_0^\infty e^{-w_i x_i}\,dx_i.
\]

<p><strong>2. Przypadek \(w_i>0\) dla każdego \(i\)</strong></p>

<p>Jeżeli</p>

\[
w_i>0
\]

<p>dla każdego \(i=1,\ldots,k\), to</p>

\[
\int_0^\infty e^{-w_i x_i}\,dx_i
=
\left[
-\frac{1}{w_i}e^{-w_i x_i}
\right]_0^\infty
=
\frac{1}{w_i}.
\]

<p>Zatem</p>

\[
\int_A f\,d\lambda_k
=
\prod_{i=1}^k\frac{1}{w_i}
=
\frac{1}{w_1w_2\cdots w_k}.
\]

<p><strong>3. Przypadek \(w_i\le0\) dla pewnego \(i\)</strong></p>

<p>Jeżeli dla pewnego indeksu \(i\) mamy \(w_i=0\), to</p>

\[
\int_0^\infty e^{-w_i x_i}\,dx_i
=
\int_0^\infty 1\,dx_i
=
+\infty.
\]

<p>Jeżeli natomiast \(w_i<0\), to</p>

\[
e^{-w_i x_i}=e^{|w_i|x_i},
\]

<p>więc również</p>

\[
\int_0^\infty e^{-w_i x_i}\,dx_i
=
+\infty.
\]

<p>W obu przypadkach, ponieważ \(f\ge0\), z twierdzenia Tonellego wynika</p>

\[
\int_A f\,d\lambda_k
=
+\infty.
\]

<p>Ostatecznie</p>

\[
\boxed{
\int_A e^{-w\cdot x}\,d\lambda_k(x)
=
\begin{cases}
\dfrac{1}{w_1w_2\cdots w_k},
&
\text{gdy } w_i>0 \text{ dla każdego } i,
\\[8pt]
+\infty,
&
\text{gdy } w_i\le0 \text{ dla pewnego } i.
\end{cases}
}
\]
