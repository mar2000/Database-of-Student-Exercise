## Treść

<p>Oblicz miarę zbioru</p>

\[
D=
\left\{
(x,y)\in\mathbb{R}^2
\mid
(x^2+y^2)^2\le 4(x^2-y^2)
\right\}.
\]

## Wskazówka

<p>Zastosuj współrzędne biegunowe \(x=r\cos\alpha\), \(y=r\sin\alpha\). Wtedy nierówność przyjmuje postać</p>

\[
r^2\le 4\cos(2\alpha),
\]

<p>co wymusza \(\cos(2\alpha)\ge 0\).</p>

## Odpowiedź

\[
4.
\]

## Rozwiązanie

<p>Przechodzimy do współrzędnych biegunowych:</p>

\[
x=r\cos\alpha,
\qquad
y=r\sin\alpha.
\]

<p>Wtedy</p>

\[
x^2+y^2=r^2
\]

<p>oraz</p>

\[
x^2-y^2
=
r^2(\cos^2\alpha-\sin^2\alpha)
=
r^2\cos(2\alpha).
\]

<p>Nierówność opisująca zbiór \(D\) przyjmuje więc postać</p>

\[
r^4
\le
4r^2\cos(2\alpha).
\]

<p>Dla \(r\ge 0\) otrzymujemy</p>

\[
r^2\le 4\cos(2\alpha).
\]

<p>Aby prawa strona była nieujemna, musi zachodzić</p>

\[
\cos(2\alpha)\ge 0.
\]

<p>Dla \(\alpha\in[0,2\pi)\) jest to równoważne</p>

\[
\alpha\in
\left[0,\frac{\pi}{4}\right]
\cup
\left[\frac{3\pi}{4},\frac{5\pi}{4}\right]
\cup
\left[\frac{7\pi}{4},2\pi\right).
\]

<p>Oznaczmy ten zbiór kątów przez \(A\). Dla \(\alpha\in A\) promień spełnia</p>

\[
0\le r\le 2\sqrt{\cos(2\alpha)}.
\]

<p>Miara zbioru \(D\) jest zatem równa</p>

\[
\lambda_2(D)
=
\int_A
\int_0^{2\sqrt{\cos(2\alpha)}}
r\,dr\,d\alpha.
\]

<p>Całkujemy najpierw względem \(r\):</p>

\[
\int_0^{2\sqrt{\cos(2\alpha)}}
r\,dr
=
\left.
\frac{r^2}{2}
\right|_0^{2\sqrt{\cos(2\alpha)}}
=
2\cos(2\alpha).
\]

<p>Zatem</p>

\[
\lambda_2(D)
=
\int_A 2\cos(2\alpha)\,d\alpha.
\]

<p>Rozpisując całkę na trzy przedziały, otrzymujemy</p>

\[
\lambda_2(D)
=
2\int_0^{\pi/4}\cos(2\alpha)\,d\alpha
+
2\int_{3\pi/4}^{5\pi/4}\cos(2\alpha)\,d\alpha
+
2\int_{7\pi/4}^{2\pi}\cos(2\alpha)\,d\alpha.
\]

<p>Ponieważ</p>

\[
\int \cos(2\alpha)\,d\alpha
=
\frac12\sin(2\alpha),
\]

<p>mamy</p>

\[
2\int_0^{\pi/4}\cos(2\alpha)\,d\alpha
=
\left.
\sin(2\alpha)
\right|_0^{\pi/4}
=
1,
\]

\[
2\int_{3\pi/4}^{5\pi/4}\cos(2\alpha)\,d\alpha
=
\left.
\sin(2\alpha)
\right|_{3\pi/4}^{5\pi/4}
=
1-(-1)
=
2,
\]

<p>oraz</p>

\[
2\int_{7\pi/4}^{2\pi}\cos(2\alpha)\,d\alpha
=
\left.
\sin(2\alpha)
\right|_{7\pi/4}^{2\pi}
=
0-(-1)
=
1.
\]

<p>Stąd</p>

\[
\lambda_2(D)
=
1+2+1
=
4.
\]

<p>Ostatecznie</p>

\[
\boxed{4}.
\]
