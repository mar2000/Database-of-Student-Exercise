## Treść

Niech
\[
\omega(x,y,z)
=
\frac{1}{z^3}\,dx
+
\frac{2y}{z^3}\,dy
-
\frac{3(x+y^2)}{z^4}\,dz,
\]
\[
p=(1,-1,6),
\qquad
q=(2,5,3).
\]

Niech \(C\) oznacza krótszy z łuków koła wielkiego sfery
\[
x^2+y^2+z^2=38
\]
o początku \(p\) i końcu \(q\).

Znaleźć całkę
\[
\int_C\omega.
\]

## Wskazówka

Spróbuj rozpoznać formę \(\omega\) jako różniczkę funkcji
\[
F(x,y,z)=\frac{x+y^2}{z^3}.
\]

Jeśli forma jest dokładna, kształt drogi \(C\) nie wpływa na wartość całki.

## Odpowiedź

\[
\boxed{
\int_C\omega=\frac{107}{108}
}.
\]

## Rozwiązanie

Rozważmy funkcję
\[
F(x,y,z)=\frac{x+y^2}{z^3}.
\]

Liczymy jej różniczkę:
\[
dF
=
\frac{\partial F}{\partial x}\,dx
+
\frac{\partial F}{\partial y}\,dy
+
\frac{\partial F}{\partial z}\,dz.
\]

Mamy
\[
\frac{\partial F}{\partial x}
=
\frac{1}{z^3},
\]
\[
\frac{\partial F}{\partial y}
=
\frac{2y}{z^3},
\]
oraz
\[
\frac{\partial F}{\partial z}
=
-\frac{3(x+y^2)}{z^4}.
\]

Zatem
\[
dF
=
\frac{1}{z^3}\,dx
+
\frac{2y}{z^3}\,dy
-
\frac{3(x+y^2)}{z^4}\,dz
=
\omega.
\]

Forma \(\omega\) jest więc dokładna na obszarze \(z\neq0\). Na rozważanym krótszym łuku \(C\) łączącym punkty
\[
p=(1,-1,6),
\qquad
q=(2,5,3)
\]
nie występuje problem z osobliwością \(z=0\).

Ponieważ
\[
\omega=dF,
\]
całka zależy wyłącznie od punktów końcowych:
\[
\int_C\omega
=
F(q)-F(p).
\]

Liczymy
\[
F(q)
=
F(2,5,3)
=
\frac{2+5^2}{3^3}
=
\frac{27}{27}
=
1,
\]
oraz
\[
F(p)
=
F(1,-1,6)
=
\frac{1+(-1)^2}{6^3}
=
\frac{2}{216}
=
\frac{1}{108}.
\]

Stąd
\[
\begin{aligned}
\int_C\omega
&=
1-\frac{1}{108}\\
&=
\frac{107}{108}.
\end{aligned}
\]

Ostatecznie
\[
\boxed{
\int_C\omega=\frac{107}{108}
}.
\]
